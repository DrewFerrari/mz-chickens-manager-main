# MZ Chickens Manager

A comprehensive poultry management system for tracking batches, storage, sales, and expenses.

## Features

- **Batch Management**: Track chicken batches from introduction to slaughter.
- **Fridge Storage**: Monitor stock levels and temperatures across multiple storage units.
- **Sales Tracking**: Record transactions with support for multiple payment methods.
- **Expense Logging**: Keep track of feed purchases and other operational costs.
- **Admin Dashboard**: Comprehensive system oversight with audit logs and data management tools.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Backend**: Supabase
- **Icons**: Lucide React
- **Notifications**: Sonner/Toast

## Local Development

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables in a `.env` file (Supabase URL and Key).
4.  Start the development server:
    ```bash
    npm run dev
    ```

## Project Identity

This project is tailored for **MZ Chickens** to streamline poultry farm operations.
